# Digikala 

This project is a clone of **Digikala**, a popular e-commerce platform in Iran. The application is built entirely with modern Android development tools, featuring a clean, responsive UI and robust architecture. It showcases a range of features found in modern shopping apps, from product browsing and searching to user authentication and a shopping cart.

## üåü Features

*   **Dynamic Home Page:** Displays promotional banners, categorized shortcuts, and special offer sections like "Incredible Offers".
*   **Product Listings:** Browse products with details like price, discounts, and availability.
*   **Navigation:** A bottom navigation bar for easy access to Home, Categories, Shopping Cart, and User Profile sections.
*   **Search Functionality:** A prominent search bar to find products across the store.
*   **Modern UI:** A clean, intuitive user interface built with Jetpack Compose that closely mimics the real Digikala app.
*   **Specialized Sections:** Includes shortcuts to services like Digikala Jet (fast delivery), DigiPay (payment service), and DigiPlus (loyalty program).

## üõ†Ô∏è Tech Stack & Architecture

This project is a showcase of modern, scalable Android development practices.

### Architecture

*   Follows the **MVVM (Model-View-ViewModel)** architectural pattern to create a clean separation of concerns and a scalable codebase.

### Key Technologies

*   **UI:** Built with **Jetpack Compose**, Google's modern declarative UI toolkit.
*   **Dependency Injection:** **Hilt** is used for managing dependencies, simplifying the codebase, and improving testability.
*   **Asynchronous Programming:** **Kotlin Coroutines** are used for managing background threads and handling asynchronous operations efficiently.
*   **Networking:** **Retrofit** and **OkHttp** are used for making network requests to a RESTful API, with a logging interceptor for debugging.
*   **Data Persistence:**
    *   **Room:** For caching data locally and enabling offline access.
    *   **DataStore:** For storing simple key-value data, such as user settings or authentication tokens.
*   **Image Loading:** **Coil** is used for loading and caching images from URLs efficiently within Compose UI.
*   **Navigation:** **Jetpack Navigation for Compose** handles all in-app navigation.
*   **UI Components & Animations:**
    *   **Accompanist:** Libraries for Swipe-to-Refresh, System UI Control, and Pagers.
    *   **Lottie:** For rendering beautiful, high-quality animations.
    *   **Vico:** For creating dynamic and stylish charts.
*   **Paging:** **Paging 3** is used to load and display large datasets incrementally.
*   **Payment:** Integrated with **Zarinpal's** payment provider library for handling transactions.
